Welcome to the Sharpfin Radio Project.

You can login to the radio (via telnet) as a:

  • guest (user/user) - RECOMMENDED
  • admin (admin/admin)
Note that it is easy to damage your radio with the admin account.
Take care!

The files for this webserver can be found in /opt/webserver

Please tell us if the installation of this patch was successfull or whether you experienced any problems using this form.
